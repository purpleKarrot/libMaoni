LibMaoni is a 3D rendering framework developed at the Visualization and
Multimedia Lab (VMML) at the Insitute of Informatics of University of Zurich UZH
as a Master Project.
.
The goal was to implement a common 3D viewer platform which can be used as
framework for the researchers' graphics and visualization projects. Requirements
were the support of different data structure for handling the 3D data and
specific rendering functions, and variable interaction capabilities.
